Mods that rely on SCS string keys may have display issues in the games' user interface with the 1.59 update. This affects some of my ETS2 mods. 

It appears that all of my currently supported mods for American Truck Simulator work without issues with the 1.59 update.

You can see all mods [here](/docs/ets2_ats/).

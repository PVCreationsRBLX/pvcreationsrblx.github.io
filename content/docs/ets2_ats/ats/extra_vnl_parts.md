This mod adds some extra parts to the 2025 Volvo VNL, primarily additional side deflector options for all sleepers. As a bonus, it also makes the sideskirt aero removable for all sideskirts.

This mod conflicts with other mods that edit these accessories:

```
std_pnt_42.volvo.vnl2025.sideskirt
std_pnt_74.volvo.vnl2025.sideskirt
std_pnt_dc.volvo.vnl2025.sideskirt
std_pla_42.volvo.vnl2025.sideskirt
std_pla_74.volvo.vnl2025.sideskirt
std_pla_dc.volvo.vnl2025.sideskirt
```
